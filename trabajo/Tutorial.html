Tutorial básico HTML5

Formato

Los documentos HTML se encuentran estrictamente organizados. Cada parte del documento está diferenciada, declarada y determinada por etiquetas específicas.

HTML usa un lenguaje de etiquetas para construir páginas web. Estas etiquetas HTML son elementos y atributos rodeados de los signos mayor y menor. La mayoría de las etiquetas HTML se utilizan en pares, una etiqueta de apertura y una de cierre, y el contenido se declara entre ellas. La de cierre se distingue por una barra invertida.

Formato de etiqueta

<elemento>

Etiqueta de cierre vs. Elemento vacío

</elemento>

Tipo de documento

En primer lugar necesitamos indicar el tipo de documento que estamos creando.

Esta línea debe ser la primera línea del archivo, sin espacios o líneas que la precedan. De esta forma, el modo estándar del navegador es activado y las incorporaciones de HTML5 son interpretadas siempre que sea posible, o ignoradas en caso contrario.

<!doctype html>

Despues de declarar el tipo de documento, debemos comenzar a construir la estructura HTML. Como siempre, la estructura tipo árbol de este lenguaje tiene su raíz en el elemento <html>. Este elemento envolverá al resto del código

<html lang="es">

El atributo lang en la etiqueta de apertura <html lang="es"> es el único atributo que necesitamos especificar en HTML5. Este atributo define el idioma del contenido del documento que estamos creando, en este caso es el español.


<!doctype html>
<html lang="es">
    <head>
        <title>Mi primer documento HTML</title>
    </head>
    <body>
        Este es el texto de mi primer documento.
    </body>
</html>

La primera sección es la cabecera y la segunda el cuerpo. El siguiente paso, por lo tanto, será crear estas dos secciones en el código usando los elementos </head> y </body>

El elemento <head> va primero y al igual que el resto de los elementos estructurales tiene una etiqueta de apertura y una de cierre:

Título

La etiqueta<title>especifica el título del documento.

Metadatos

Algunos cambios e innovaciones fueron incorporados dentro de la cabecera, y uno de ellos es la etiqueta que define el juego de caracteres a utilizar para mostrar el documento. Ésta es una etiqueta <meta> que especifica cómo el texto será presentado en pantalla:


<!doctype html> 
<html lang="es">
<head>   
<title>Mi primer documento HTML</title>
   <meta charset="utf-8">
   <meta name="author" content="Rafa">
  </head>
  <body>
      Este es el texto de mi primer documento.
  </body>
</html>

Hay varios tipos de etiqueta <meta> que pueden ser incluidas para declarar información general sobre el documento, pero esta información no es mostrada en la ventana del navegador, es solo importante para motores de búsqueda y dispositivos que necesitan hacer una vista previa del documento u obtener un sumario de la información que contiene. Aparte del título y algunos íconos, la mayoría de la información insertada entre las etiquetas <head> no es visible para los usuarios

Párrafo

PARAGRAPH: P

La marca < p> sirve para definir un nuevo párrafo de texto, indicando al navegador que el mismo debe empezar en una nueva línea

    < p>Este texto pertenece a un párrafo.</p>
    < p>Pero éste es del párrafo siguiente</p>

Texto preformateado

PREFORMATTED: PRE

La etiqueta <pre> define el texto preformateado. El texto de un elemento <pre> se muestra en una fuente de anchura fija (normalmente Courier) y conserva espacios y saltos de línea. 
    
<pre>                Este texto aparecerá
     escrito en el navegador con este mismo formato
                       utilizando un tipo de letra monoespaciada. 
</pre>

Listas

Elementos

Unordered List: ul
Ordenadas
Ordered List: ol
Definition List: dl

Definition Data: dd

La etiqueta <ul>define una lista desordenada (con viñetas).

La etiqueta <ol> define una lista ordenada. Una lista ordenada puede ser numérica o alfabética.

Se utiliza la etiqueta <li> para definir los elementos de la lista.

La etiqueta <dl> define una lista de descripción. La etiqueta <dl> se utiliza conjuntamente con <dt> (define términos / nombres) y <dd> (describe cada término / nombre).

Línea horizontal

HORIZONTAL RULE: HR

La etiqueta <hr> define una ruptura temática en una página HTML (por ejemplo, un cambio de tema).

    <p>La siguiente línea horizontal divide temáticamente este párrafo del siguiente</p>
    <hr>
    
    <p>Este segundo párrafo puede pertenecer, por ejemplo, a un cambio de escena dentro de un mismo capítulo de una novela.</p>
